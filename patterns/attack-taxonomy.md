# Attack Taxonomy (Defensive)

## 1) Prompt Injection
- instruction override
- context confusion
- indirect injection via documents

## 2) Jailbreak
- roleplay / hypothetical framing
- multi-turn persuasion
- obfuscation tricks

## 3) Data Leakage
- system prompt leaks
- memory leaks
- RAG overexposure
- tool-output leaks

## 4) Tool Abuse
- unauthorized tool execution
- argument injection
- tool output injection

## 5) Supply Chain / Integration
- unsafe plugins
- unreviewed connectors
