# Scope & Ethics

This repository is designed for **defensive security** and **authorized testing** of AI systems.

## Allowed Use
- Testing systems you own
- Testing systems you have explicit permission to test
- Building safer prompts, safer tool-use, safer RAG pipelines
- Evaluating AI output quality and security

## Not Allowed Use
- Attempting unauthorized access to systems/data
- Guidance for real-world wrongdoing or harm
- Instructions to exfiltrate secrets from third-party systems

## Principles
- Minimize sensitive data exposure in prompts/logs
- Use least privilege for tools and data sources
- Prefer test environments and synthetic data
- Document findings and mitigations responsibly
