# Glossary

- **LLM app**: Any application that uses a language model to generate outputs.
- **Prompt injection**: Adversarial input that tries to override app instructions.
- **Jailbreak**: Attempts to bypass policy restrictions or safety behavior.
- **RAG**: Retrieval-Augmented Generation — model uses retrieved documents as context.
- **Tool use / Agents**: Model can call tools (search, DB, email, filesystem, etc.).
- **Data exfiltration**: Unauthorized extraction of secrets/sensitive data.
- **Guardrails**: Mitigations that reduce unsafe or insecure behavior.
